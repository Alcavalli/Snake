# Snake

Implementazione del classico gioco Snake in C++ con l'uso di SFML


## Dipendenze

- CMake 3.10+
- SFML 2.6


## Come compilare

```bash
mkdir build
cd build
cmake ..
cmake --build .
```


## Come giocare

- Frecce direzionali per muovere il serpente
- Mangia il cibo rosso per crescere
- Evita i bordi e te stesso


## Struttura

```
src/        # sorgenti e header
assets/     # font e risorse grafiche
```
